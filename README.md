# 🎧 DTRX Audio Lab

Interactive web experience created to combine music production and web development.

## 🚀 About the Project

DTRX Audio Lab is an experimental project that connects an interactive audio experience with my music channel.

The goal is to create engaging tools that attract users through sound interaction and naturally lead them to discover my music.

## 🎧 Features

- Headphone test (stereo panning L → R)
- Drop trigger button (interactive audio effect)
- Embedded music player (YouTube)
- Mobile-first responsive design

## 🎵 About My Music

I produce emotional and cinematic EDM focused on dark atmospheres and powerful drops.
